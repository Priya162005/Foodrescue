# FoodRescue

A static food rescue website built with HTML, CSS, and JavaScript.

## Run locally

Open `index.html` in your browser, or use a local server such as Live Server in VS Code.

## Deployment

### GitHub Pages

1. Enable Pages on branch `gh-pages` and folder `/`.
2. Wait a few minutes for the site to publish.
3. The URL should be:
   - `https://Priya162005.github.io/Foodrescue/`

### Netlify

1. Connect your GitHub repo to Netlify.
2. Set the publish directory to `FoodRescue`.
3. Use the existing `netlify.toml`.

This repository is ready for static deployment.
